SafeK

외국인 관광객을 위한 시인성 극대화 및 지능형 재난 대피 네비게이터.
최종 서비스 검토 및 핏칭(Pitching) 덱 문서입니다.

최종 서비스 검수 및 점검

1. 아키텍처 및 배포 환경

인프라 하이브리드 지원: Docker Compose (Local/Dev) 및 Kubernetes (Production Scale) 이원화 구성이 완료되었습니다.

CI/CD 자동화: `.github/workflows/main-ci.yml` 파이프라인을 통해 GitHub -> DockerHub 이미지 푸시 자동화가 적용되어 있습니다.

모니터링: Prometheus + Grafana를 연동해 실시간 시스템 부하율을 측정합니다.

2. 안정성 확보 (Test Logs)

FastAPI 기반의 최적화된 백엔드는 시스템 핵심 컴포넌트에 대한 통합 테스트를 통과했습니다.

❯ python -m pytest tests/test_api.py
================= test session starts =================
collected 3 items
tests\test_api.py ... [100%]
======== 3 passed in 0.78s =========

프론트엔드 React 역시 `ESLint` 검증에서 `No errors or warnings!`로 무결성을 획득했습니다.

UI/UX 최종 디자인 시안 (디자인 철학)

부드러운 다크톤과 극한상황에서의 시인성(고대비)을 동시에 챙긴 "SafeK"만의 시그니처 룩(Signature Look)입니다.

메인 대시보드 (Home View)

직관적이고 굵은 폰트를 사용한 실시간 호우 경보 알림 배너. 주변 안전 지대를 요약하여 보여줍니다.

긴급 SOS (SOS View)

다국어 지원 토글과 함께 오작동 확률이 0에 수렴하는 글로잉 매크로 버튼. 실시간 GPS 좌표를 표출합니다.

안전 네비게이터 (Route View)

침수/위험 지역은 붉은 점선으로 차단하고 푸른색(청색) 네온 라인으로만 안전한 경로를 추천하는 AI 연동 맵 시스템.

SafeK 인터랙티브 영상 데모 (시연)

시연 리허설 스크립트

  • 도입: "저희 SafeK 모바일 웹 화면입니다. 상단에 가장 시급한 기상 특보가 보입니다."
  • 글로벌/현지화: "상단 언어 토글을 클릭해 한국어로 실시간 변환되어 외국인 접근성을 높였습니다."
  • SOS 증명: "실제 재난 발생 시 긴급 상황 메뉴에서 거대한 SOS 버튼 한 번으로 119/대사관에 GPS를 송출합니다."
  • 정리: "이 모든 UX는 도커(Docker) 기반 또는 K8S를 기반으로 운영되며, 현재 서버 환경 없이 브라우저 단에서 매끄러운 렌더링 검수에 성공했습니다."
